项目简介

简介

EmeraldLib 前置插件提供的 API。插件使用了 Kotlin 语言，提供了诸多对于插件编写上语言层面的优化。本项目引入了 spigot-1.12.2 作为依赖，对于 Craft 和 Nms 的操作均使用 ASM Reflection 完成。

内容

API 包含以下大致内容：

  • Dsl语言结构注册命令
  • Dsl语言结构注册权限
  • Dsl语言结构注册事件监听器
  • Dsl语言结构物品NBT属性修改
  • Location,Vector 运算符重载
  • ChatComponent 对于 md_5 实现的扩展。
  • TextComponent 的轮子
  • Scheduler 的顶层函数
  • Player 的扩展
